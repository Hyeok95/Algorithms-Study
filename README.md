## ✅ 참여 방법
1. 이 저장소를 fork 한다.
2. 생성된 원격 저장소에 이름으로 폴더를 생성한다.
3. 생성된 폴더에 자신의 소스코드를 업로드 한다.
4. 매주 월요일에 본인이 작성한 소스코드를 원본 저장소로 fetch merge를 한다.
